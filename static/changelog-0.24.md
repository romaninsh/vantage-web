# Vantage 0.24

## What's new

- **Browse a tree like a file manager.** A new `finder` component turns a path-shaped source
  into a Finder-style browser: a breadcrumb and an expandable folder tree beside the selected
  folder's listing, with byte-humanized sizes and Finder-style timestamps. Expanding a folder
  opens just that path's data and collapsing closes it, so deep trees stay light — and pointed
  at a live source the listing moves on its own as files come and go. The selected path is
  published for the rest of the page to read.
