# Vantage Framework Website

Static site built with [Zola](https://www.getzola.org/).

## Local Development

1. **Install Zola**

   ```bash
   # macOS
   brew install zola

   # Or download from https://github.com/getzola/zola/releases
   ```

2. **Run locally**

   ```bash
   zola serve
   ```

Site will be available at `http://127.0.0.1:1111`

## Deployment

Automatically deploys to GitHub Pages on push to `main` branch.
